Simulation of Summary Statistics

  1. Sample host tree η.

    Inputs

    • Sequence data.

    • Divergence time calibration priors.

    Outputs

    • Ultrametric phylogeny.

    Remarks

    • Use BEAST to sample from the posterior distribution of trees given the data.

  2. Sample augmented host tree η*, conditioned on host tree, η.

  3. Extract schedule of events from augmented host tree, η*.

  4. Simulate symbiont tree, ψ, conditioned on schedule of events on host tree.

  5. Scale symbiont tree by each substitution rate scaling factor to yield scaled symbiont tree ψ′.

  6. Calculate summary statistics on scaled symbiont tree, ψ′.
